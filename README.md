✅ PART 1: Project Name
# Error Insight Pro

✅ PART 2: One-line Description
A full-stack Python web application that analyzes runtime error messages and provides clear explanations.

✅ PART 3: Features
## Features
- Analyzes error messages
- Explains common runtime errors
- Stores error history
- Built using Flask and SQLite

✅ PART 4: Tech Stack
## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS

✅ PART 5: How to Run
## How to Run
pip install flask
python app.py
Open http://127.0.0.1:5000/
